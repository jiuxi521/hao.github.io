序列化的类需要满足以下条件：  
1. 实现Serializable接口  
2. 自定义serialVersionUID，如果不自定义，会默认创建一个序列化值；{@code private static final long serialVersionUID = 1L;}  
3. 静态属性和transient修饰的属性默认不会被序列化  
序列化类支持重写  
1. void writeObject(ObjectOutputStream oos)：自定义序列化器  
2. void readObject(ObjectInputStream ois) ：自定义反序列化器  
3. Object readResolve() ： 重写反序列化后的对象，可用于单例模式中单例对象的反序列化，保证唯一性  
4. Object writeReplace()： 重写要序列化的对象，可用于代理对象的序列化，使用代理对象的属性创建一个新对象，用于序列化  
4. void readObjectNoData()： 可用在 对象不存在序列化值的反序列化场景中，给对象赋默认值    {@link test.serial.BaseClass#readObjectNoData()}
